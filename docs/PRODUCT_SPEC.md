# Especificação do produto

## Propósito

O site apresenta o trabalho de Thales Santos e seu posicionamento como desenvolvedor PHP / Full Stack com foco em backend, sustentação e evolução de sistemas.

## Conteúdo

A página reúne:

- apresentação profissional e principais tecnologias;
- projetos com links para código e demonstrações, quando disponíveis;
- competências técnicas;
- informações profissionais e formas de contato;
- uma experiência com dados da API APOD, da NASA.

## Experiência

- O conteúdo é organizado em uma página única com navegação por seções.
- O Hero comunica o posicionamento profissional e oferece acesso direto aos projetos e à experiência NASA.
- A interface é responsiva e usa animações como apoio visual.
- A integração NASA permite consultar o conteúdo astronômico de uma data válida.

## Limites do repositório

- A aplicação é exclusivamente client-side.
- Não há backend ou persistência de dados neste repositório.
- A publicação é feita como site estático no GitHub Pages.
